Execicio 01 - DIO

Criar uma aplicação que resolva as seguintes situações:

1 -  calcule as 4 operações básicas: 

	Soma
	Subtração
	Multiplicação
	Divisão
	
<kbd>OBS: Não consegui que a variável operação lê-se a String "+", "-", "*", "/". Por isso criei um menu para escolher a operação.</kbd>

2 -  A partir da hora do dia, informe a msenagem adequada:

	Bom Dia
	Boa Tarde
	Boa Noite
	
3 - Calculoe o valor final de um emprestimo,a  aprtir do valor solictado. Taxas e parcelas influenciam. Defina arbitrariamente as faixas que inflienciam nos valores.
